# 交易行平台规则文档
> 本文档为 AI 客服 RAG 知识库。所有规则均为虚构，用于 Spring AI 学习项目。

---

## 一、用户注册与身份认证

### 1.1 注册资格
- 仅限全国高校在校大学生、研究生、博士生（含交换生）注册
- 校友（毕业 2 年内）可保留账号，2 年后降级为"游客"模式
- 每位用户仅可注册 **1 个**主账号，学号/身份证号/手机号任一重复均拒绝注册

### 1.2 校园邮箱认证
- 支持以下后缀：`.edu.cn`、`[学校名].edu.cn`、`[学校名].edu`
- 其他邮箱后缀一律拒绝注册
- 同一学校邮箱后缀不可重复注册

### 1.3 实名认证（学信认证）
- 需提交：真实姓名、学号、学生证截图（需含学号和注册日期）
- 人工审核窗口：24 小时内完成
- 审核拒绝原因将通过站内信告知，可申诉 1 次

### 1.4 登录安全
- JWT Token 有效期：7 天
- 刷新 Token 有效期：30 天
- 异地登录（IP 段变更）触发短信二次验证
- 连续 5 次密码错误锁定账号 30 分钟

### 1.5 账号注销
- 注销需经过 30 天冷静期（期间可撤销）
- 冷静期内账号功能冻结，但聊天记录保留
- 30 天后物理删除所有个人数据（学号、身份证、联系方式）

## 二、商品发布规则

### 2.1 分类体系（3 层深度）

**一级分类（14 类）：**
1. 数码电子 / 2. 图书教材 / 3. 服饰鞋包 / 4. 美妆护肤 / 5. 日常生活 / 6. 运动户外 / 7. 虚拟物品 / 8. 校园服务 / 9. 考研专区 / 10. 毕业季专区 / 11. 失物招领 / 12. 拼车返乡 / 13. 代取快递 / 14. 其他

**二级分类示例：**
- 数码电子 → 手机 / 电脑 / 平板 / 耳机/音响 / 相机/镜头 / 充电器/线材 / 存储设备
- 图书教材 → 理工科 / 文科类 / 考研资料 / 四六级 / 考公资料 / 笔记/手写稿
- 考研专区 → 初试资料 / 复试资料 / 导师信息 / 经验分享

### 2.2 标题规范
- 字数限制：5~30 中文字符
- 禁止词汇（12 个诱导词）：急出 / 血亏 / 自刀 / 倒贴 / 甩卖 / 骗子 / 赔钱卖 / 跳楼价 / 免费送 / 仅此一天 / 错过后悔 / 限量
- 标题需包含商品名称、规格（如有）、成色
- 示例：`【二手】iPad Pro 2022 128G 9成新 带笔`

### 2.3 价格规则
- 价格区间：0.1 元 ~ 99,999 元
- 若开启"可议价"标签，议价底价不得低于原价的 30%
- 故意标价 0.01 元吸引流量视为违规
- 不得以"定金""意向金"名义收取费用

### 2.4 图片规范
- 图片数量：1~9 张
- 单张大小：≤ 5MB
- 支持格式：JPG / GIF / PNG / WebP
- **必须去除 EXIF 定位信息**（平台自动处理，用户无权干涉）
- 禁止盗用他人图片，违者下架并扣信用分

### 2.5 商品描述规范
- 字数：10~500 中文字符
- Emoji 上限：20 个
- 必须包含：成色说明、使用情况、转手原因、适合人群
- 禁止：夸大宣传、虚构成色、承诺保证

### 2.6 禁售品类（47 类，AI 语义识别重点）

**绝对禁止（违者永久封号）：**
1. 烟、酒（含啤酒/白酒/葡萄酒）

5. 色情低俗物品

- 满 50 元全国包邮（平台补贴运费）
- 专区商品优先推荐给同校买家

### 2.8 考研专区
- 仅允许：二手教材 / 考研笔记 / 录音 / 经验分享
- **禁止**：考研真题答案、报考信息买卖、导师"打招呼"服务
- 发布考研资料需标注来源（自用/购买/赠予）

## 三、交易与支付规则

### 3.1 资金监管
- 买家付款后，资金进入平台监管账户
- 确认收货后 T+1 工作日自动打款至卖家账户
- 平台不抽佣，资金 100% 归卖家

### 3.2 交易方式
| 方式 | 说明 | 费用 |
|---|---|---|
| 面交（同校） | 校园内当面交易 | 免费 |
| 面交（异校） | 线下见面交易 | 免费 |
| 邮寄 | 快递运输 | 买家承担（卖家包邮除外） |
| 顺丰到付 | 额外 +5 元服务费 | 买家承担 |

### 3.3 收货确认窗口
- 物流签收后 7 天内未操作 → **自动确认收货**
- 超时未确认，买家可申请延期 3 天（每月限 1 次）
- 争议期间资金冻结，不打款给卖家

### 3.4 退款流程（5 步）
1. 买卖双方自行协商
2. 买家发起退款申请（附理由）
3. 买家上传凭证（照片/视频/聊天记录）
4. 平台客服 48 小时内介入
5. 平台判决：一刀切（全额退/部分退/驳回）

### 3.5 争议仲裁金额阈值
| 金额区间 | 处理方式 |
|---|---|
| < 200 元 | 不介入人工，直接按规则判决 |
| 200 ~ 2000 元 | 后台客服人工介入 |
| > 2000 元 | 提交校园仲裁委员会（每学期开庭 2 次） |

### 3.6 卖家发货规范
- 需在 48 小时内发货（上架/物流单号）
- 超时视为"拖延发货"，扣信用分 -3
- 邮寄商品需拍照留证（打包前、打包后、快递员取件）
- 顺丰到付额外 +5 元服务费由买家承担

## 四、信用分体系

### 4.1 等级划分

| 等级标识 | 等级名 | 信用分区间 | 权益 |
|---|---|---|---|
| ⭐ S0 | 新星 | 0 ~ 59 | 基础权限，商品曝光最低 |
| ✅ S1 | 可信 | 60 ~ 74 | 普通权限，可发布毕业季商品 |
| 👍 S2 | 良好 | 75 ~ 89 | 推荐加权，优先展示 |
| 🌟 S3 | 优秀 | 90 ~ 96 | 无理由退（需买家承担运费） |
| 💎 S4 | 钻石 | 97 ~ 100 | 全站金牌商家标识，客服优先通道 |

### 4.2 加分项

| 行为 | 信用分变化 | 说明 |
|---|---|---|
| 成功完成一笔交易（买/卖） | +1 | 每笔限 1 次 |
| 成功售出一件商品 | +2 | 鼓励卖出 |
| 主动给好评 | +0.5 | 仅买家可操作 |
| 收到好评 | +1 | 双向好评 |
| 申诉成功（违规判定撤销） | +10 | 需提供充分证据 |
| 连续 30 天登录 | +2 | 不限次数 |
| 举报他人违规（核实） | +3 | 每月限 3 次 |
| 参加平台活动（开学季/毕业季） | +5 | 每年限 2 次 |

### 4.3 减分项

| 行为 | 信用分变化 | 备注 |
|---|---|---|
| 虚假商品（第一次） | -10 | 下架商品 |
| 虚假商品（第二次） | -30 | 7 天禁发 |
| 虚假商品（第三次） | -70 | 30 天封号 |
| 爽约面交（双方确认） | -15 | 需对方举证 |
| 恶意差评（平台判定） | -5 | 被识破才扣 |
| 拖延发货 | -3 | 48h 内未发 |
| 诱导私下交易 | -20 | 严重违规 |
| 申诉被驳回 | -3 | 滥用申诉通道 |
| 30 天无登录 | -1 | 按天累计 |
| 发布违禁品 | -50 起 | 视品类决定上限 |
| 触发风控熔断 | -100 | 自动降级至 S0 |

### 4.4 信用分恢复
- 自然恢复：每天 +0.5（需无新增违规）
- 被降级至 S0 后，需完成 5 笔好评交易才能申请 S1 复审
- 被封号（-100）30 天后可申请解封，需人工审核

## 五、违规与处罚分级

### 5.1 违规等级

| 等级 | 名称 | 触发条件 | 处罚 | 申诉 |
|---|---|---|---|---|
| L0 | 警告 | 首次轻微违规 | 站内私信通知 | 不可申诉 |
| L1 | 轻罚 | 诱导性标题/未分类/图片不规范 | 下架商品 | 可申诉 |
| L2 | 中罚 | 虚假价格/虚假图片/拖延发货 | 7 天禁发商品 | 可申诉 |
| L3 | 重罚 | 假冒他人商品/欺诈未遂 | 30 天封号 | 不可申诉 |
| L4 | 永封 | 重大违法/套现平台资金/恶意刷单 | 永久封号+上报校方 | 不可申诉 |

### 5.2 违规累积规则
- L0 累积 3 次自动升级为 L1
- L1 累积 2 次自动升级为 L2
- L2 累积 2 次自动升级为 L3
- L3 出现即升级 L4

### 5.3 申诉规则
- L0 警告不可申诉
- L1、L2 申诉期限：违规通知后 7 天内
- 申诉通过条件：用户提供充分证据证明无误判
- 申诉通过后：信用分 +10（参照 4.2），违规记录清除
- 申诉驳回后：30 天内不得再次申诉同一违规

### 5.4 处罚执行细则
- 下架商品：立即执行，用户可编辑后重新上架（需审核）
- 禁发：禁止发布新商品，旧商品不受影响
- 封号：全部功能冻结，聊天记录保留，30 天后可申请解封（L3）
- 永封：全部功能永久关闭，数据移交学校保卫处（涉嫌违法时）

## 六、隐私与数据规则

### 6.1 信息展示
- 手机号展示格式：`180****5678`（前 3 后 4 明文，中间星号）
- 学号：**不展示**，后台加密存储
- 真实姓名：仅展示姓氏，如"张同学"
- 宿舍/班级：**不展示**

### 6.2 聊天记录
- 保留期限：180 天
- 期满后自动清除（不含已下单的交易记录）
- 涉及违规举报的记录永久保留至案件结案

### 6.3 登录安全
- 校园网（.edu.cn IP 段）：无限制
- 校外 IP 登录：触发短信验证码
- 连续 5 次登录失败：锁定 30 分钟

### 6.4 数据删除
- 账号注销 30 天冷静期后物理删除
- 删除范围：个人资料、学号、身份证、联系方式、收货地址
- **保留**：交易记录（财务合规）、违规记录（平台治理）

## 七、特殊业务专区规则

### 7.1 毕业季专区
- 开放时间：每年 3/1-7/31、9/1-9/30
- 仅 S1+ 毕业生可发布"出清"标签
- 满 50 元全国包邮（平台补贴上限 15 元/单）
- 专区商品优先推送给同校用户

### 7.2 考研专区
- 仅允许：二手教材、考研笔记、录音、经验贴
- 禁止：考研答案、代导师"打招呼"、报考信息买卖
- 资料来源标注：自用 / 购买 / 赠予 / 网购

### 7.3 校园跑腿
- 同校接单，禁止跨校
- 最低单价：3 元
- 平均时薪上限：15 元（防恶意竞争）
- 平台提供基础保险（每单 0.5 元保费，保额 200 元）

### 7.4 拼车返乡
- 仅允许寒假（1-2 月）/ 暑假（7-8 月）
- 驾驶员要求：持证 1 年以上 + 行驶证 + 驾照
- 每车次限拼 3 人（不含驾驶员）
- 平台代收定金（≤ 20 元），行程结束后确认退款

### 7.5 失物招领
- 发布后 24 小时内仅校内可见
- 超过 24 小时自动转公区（全校可见）
- 超过 30 天无人认领，物品归平台处置（销毁或捐赠）

### 7.6 代取快递
- 单笔费用上限：3 元
- 接单距离：≤ 1km（按校园地图直线距离）
- 每位用户每天最多发布 5 单代取需求
- 接单者需在 2 小时内完成，违者扣信用分 -2

### 7.7 二货市集（线下）
- 仅限经学校审批通过的线下跳蚤市场活动
- 报名资格：学校社团联合会审批 → 平台认证
- 每年每校区最多举办 2 次
- 活动期间平台提供线上直播展示（可选）

## 八、平台运营策略

### 8.1 新人福利
- 注册即送 5 元无门槛优惠券（限首单使用）
- 首单完成后返 3 元积分（下一次购物抵扣）
- 新人商品前 7 天曝光加权

### 8.2 信用分级权益
- S3+ 可申请"无理由退"（买家承担运费）
- S4 享有客服优先通道（排队优先）
- S0 用户商品曝光权重最低，搜索结果排在最后

### 8.3 推荐算法策略
- 屏蔽：S0 用户商品（信用过低）
- 屏蔽：违禁品类（自动过滤）
- 加权：同校商品（S2+ 加权 1.5 倍）
- 加权：近期发布（72 小时内加权）
- 降权：长期未售出（30 天后降权 50%）

### 8.4 月度活动
- 每月评选"校园之星"：按城市分赛区，信用分 + 交易量综合排名
- 前 3 名额外获得首页 Banner 推荐 7 天
- 开学季（3/9 月）：新人免手续费活动

### 8.5 红包雨
- 重大节日（元旦/春节/开学/毕业）：平台发放随机红包
- 红包仅限平台内使用，不可提现
- 红包有效期：7 天

## 九、AI 客服能力范围

### 9.1 可回答的问题类型
1. **平台使用**：如何发布商品、如何搜索、如何下单、如何联系卖家
2. **交易规则**：议价规则、发货时效、退款流程、面交注意事项
3. **信用分**：我有多少分、怎么涨分、被扣分了怎么办
4. **违规处罚**：为什么被下架/禁言、如何申诉
5. **账户安全**：密码找回、异地登录验证、账号注销
6. **特殊业务**：毕业季专区如何发布、考研资料规则、拼车规则

### 9.2 回答规则
- 每次回答不超过 150 字
- 口语化，称呼用户"同学"
- 不确定时回复"我帮您转人工客服"
- 禁止编造平台不存在的功能
- 禁止承诺保证（如"一定能退款"）

### 9.3 超出能力范围（转人工）
- 涉及资金损失 > 200 元的纠纷
- 涉及人身安全的问题
- 举报违法/色情/暴力内容
- 需要查询具体订单状态（提供订单号后转人工）
- 用户情绪激动/投诉升级
